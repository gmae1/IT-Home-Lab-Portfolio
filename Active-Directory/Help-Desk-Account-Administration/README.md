# Help Desk Account Administration

## Objective
Perform common Active Directory account administration tasks including password reset, forced password changes, account disabling, account enabling and group membership verification.

## Why This Lab Matterws
Account administration is one of the most common reponsibilities of Help Desk Techs, Desktop Support Specialist, and Junior System Administrators. Proper user lifecycle management helps maintain security and ensures users have appropriate access to company resources.

## Task Performed
- Reset User Password
- Verified Account Status
- Simulated Employee Termination
- Simulated Employee Rehire
- Tested Authentication Failures
- Verified Group Membership

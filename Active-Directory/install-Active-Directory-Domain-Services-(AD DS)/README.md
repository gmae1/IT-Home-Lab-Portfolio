# Install Active Directory Domain Services (AD DS)

## Objective
Install the Active Directory Domain Services role and promote Windows Server 2022 in the first Domain Controller for the lab environemt.

## Tools Used

- Windows Server 2022
- Server Manager
- Active Directory Domain Services
- DNS
- Command Prompt

## Task Performe

-Installed the Active Directory Domain Services role
- Added required management tools
- Promoted DC1 to a Domain Controller
- Created a new forest
- Created the domain LAB.LOCAL
- Configured DNS during domain creation
- Created a Directory Service Restore Mode (DSRM) password
- Verified domain functionality

## What I Learned

- Active Directory Domain Services provides centralized management of users, computers, and policies
- A Domain Controller host Active Directory and authentication services.
- Domains contain users, computers, groups and polocies
- DNS is automatically integrated with Active Directory
- The DSRM password provides recovery capabilities for Active Directory
- Domain Controllers are foundational components of enterprise networks.

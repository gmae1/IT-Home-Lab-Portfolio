# Active Directory Users and Organizational Units Lab

## Objective
Create Organizational Units (OUs) and user accounts to logically organize departmental resources within the Active Directory domain.

## Why This Lab Matters
Organizational Units provide structure within Active Directory and allow administrators to logically organize users, computers, and resources. Proper organization simplifies administration and enables Group Policy deployment to specific departments/

## Task Performed
- Open Active Directory Users and Computers
- Created Organizational Units
  - HR
  - Sales
  - IT
  - Engineering
  - Servers
- Created User Accounts
- Configured Passwords for All Users
- Verified OU Structure

# DHCP Services and IP Address Management

## Objective 
Deploy and configure DHCP services to automatically assign IP addresses and centrally manage network addressing through scopes and reservations

## Why This Lab Matters
DHCP eliminates the need to manuallu configure IP addresses on every device. Centralized IP management simplifies administration and ensures devices receive consistent network configurations.

## Task Completed
- Installed DHCP Role
- Completed DHCP Configuration
- Created DHCP Scope
- Assigned a Address Pool Range
- Assigned Lease Duration
- DNS Comfiguration
- Obtained Machines MAC Address
- Created Reservation
- Verified the Reservation
- Verified the IP Address

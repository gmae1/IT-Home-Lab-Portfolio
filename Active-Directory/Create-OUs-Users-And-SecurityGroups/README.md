# Create Organizational Units, Users and Security Groups Lab

## Objective
Build a basic Active Directory company structure by creating OUs, User Account and Security Groups for Different Departments

## Tools Used
- Windows Server 2022
- Active Directory User and Computers
- Active Directory Domain Services
- Security Groups

## Task Performed 
- Opened Active Directory User and Computers
- Created Organizational Units (OUs):
    - IT
    - HR
    - Sales
    - Engineering
- Created user accounts for each departments
- Created security groups for each department
- Added users to their respective groups
- Verified group membership

## What I Learned

- Organizational Units are used for organization
- Security Groups are used to assign permissions
- Users inherit permissions from groups they belong to
- Active Directory automatically places users into Domain Users
- Group-Based administration scales much better than assigning permissions to individual users
- OUs and Groups Server different purposeses

# Group Policy Resource Deployment Lab

## Objective
Use Group Policy Preferences to automatically deploy network drives to users based on Active Directory security group membership

## Why This Lab Matters
Organizations use Group Policy to provide users with automatic access to department resources without manually configuring workstations

## Task Performed 
- Created Test File
- Created Group Policy Object
- Configured Drive Mapping
- Configured Item-LEvel Targeting
- Applied Group Policy
- Verified Drive Mapping

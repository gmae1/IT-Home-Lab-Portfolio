# Group Policy Workstation Restriction Lab

## Objective
Implement user-based Group Policy Objects (GPOs) to restrict access to Control Panel and Windows Settings, preventing unauthorized changes to workstation cnfigurations

## Why This Lab Matters
Organizations use Group Policy to limit user access to critical system settings in order to:
  - Maintain configuration consistency
  -  Prevent accidental changes
  -  Reduce Help Desk Tickets
  -  Improve endpoint security

This allows administrators to manage workstations centrally without manually cconfiguring every device.

## Task Performed

- Created New Group Policy
- Edited the Group Policy
- Applied group Policy
- Verified Restriction

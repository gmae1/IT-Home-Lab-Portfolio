# Powershell Active Directory Administration Lab

## Objective
Use Powershell to automate common Active Directory administration task, including querying users, creating accounts, bulk importing users from CSV files, and disabling accounts

## Why This Lab Matters
Powershell allows administratiors to manage Active Directory at scale instead of manually clicking through ADUC. Automation reduces errors and saves time when managing dozens of hundreds of account.

## Task Performed 
- Made CSV in Notepad
- Disabled Bulk Accounts
- Verified Disabled Accounts
- Reset Passwords by the Bulk
- Queried User Information

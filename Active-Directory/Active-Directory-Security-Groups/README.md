# Active Directory Security Groups Lab

## Objective
Create Global Security Groups and assign user to those groups to simplify permission management and support role-based access control

## Why This Lab Matters
Security Groups provide a scalable way to manage permissions. Instead of assiging access directly to users, administrators assign permissions to groups and add users to those groups. This approch simplifires administration and follows of the Princple of Least Privlege.\

## Task Performed
- Open Active Directory Users and Computers
- Created Global Security Groups
- Added Users to Their correct Groups
- Verified Group Membership

# Endpoint Security Policies Lab

## Objective
Implement endpoint hardening policies through Group Policy to restrict removable storage devices and prevent unauthorized use of Command Prompt

## Why This Lab Matters
Organizations use Group Policy to enforce endpoint security controls and reduce attack surfaces by limiting user access to potentially dangerous tools and device.
These policies help protect systems against malware, data exfiltration and unauthorized administratice actions.

## Task Performed
- Create Group Policies
- Configure Policy
- Applied Policy
- Verified Restriction

# Objective
Configure DNS services in Active Directory and join a Windows 11 Pro Workstation to the LAB.LOCAL domain to enable centralized authentication and management

## Why This Lab Matters
Active Directory relies heavily on DNS to locate Domain Controllers and services required for authentication. Joining a workstation to the domain enables centeralized account management, Group Policy, and access to shared resources.

## Task Performed
- Verified DNS Integration
- Test DNS Resolution
- Test Name Resolution
- Configured DNS on Windows 11 Pro
- Joined Windows 11 Pro to the Domain
- Restarted and Logged into Windows 11 using Administrator Profile
- Verified Domain Membership
- Verified Computer Object Creation

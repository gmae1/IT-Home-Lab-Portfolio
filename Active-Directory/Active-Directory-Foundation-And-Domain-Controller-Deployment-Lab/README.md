# Active Directory Foundation and Domain Controller Deployment Lab

## Objective
Deploy Windows Server 2022 and configure it as a Domain Controller by installing Active Directory Domaine Services and creating the LAB.LOCAL forest.

## Why This Lab Matters
Active Directory provides centeralized authenitcation and management for users, computers and resources. Domain Controllers allow organizations to manage hunderds or thousands of devices from a single location.

## Task Performed
- Renamed the Server to DC1
- Configured the Servers 2nd Adapter (Internal Adapter) IP Address to a static IP: 192.168.50.10
- Installed Active Directery Domain Services
- Promoted Server (DC1) to Domain Controler
- Verified NetBIOS Name
- Verified Successful Promotion

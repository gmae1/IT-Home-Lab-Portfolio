# Command Promt Essential Lab

## Objectives
Practice using common Command Prompt tools to gather system information, verify network connectivity, and troubleshoot Windows systems.

## Tools Used
- Windows 11
- VirtualBoc
- Command Porompt

## Task Performed
- Gathered system information
- Identitfied computer hostname
- Reviewed network configuration
- Tested network connectivity
- Verified DNS name resolution
- Traced network ping
- Investigated running processes
- Reviewed active network connection

  ## Commands Used

  ### cmd
  - hostname
  - systeminfo
  - ipconfig
  - ipconfig /all
  - ping 8.8.8.8
  - ping google.com
  - tracert google.com
  - tasklist
  - netstat -ano

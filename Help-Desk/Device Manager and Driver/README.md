# Device Manager and Driver Lab

## Objective
Practice investigating hardware devices and drivers using Windows Device Manager.

## Tools User
- Windows 11
- VirtualBox
- Device Manager

## Task Performed
- Opened Device MAnager
- Explored hardware categories
- Investigated device properties
- Reviewed device properties
- Examined device status
- Scanned for hardware changes
- Practiced basic device troubleshooting procedures

## What I Learned
- How Windows organizes hardware devices in Device Manager
- How to view device properties and status information
- How to locate driver details such as version and provider
- How Device Manager is used to troubleshoot hardware issues
- How to scan for hardware changes during troubleshooting
- Importance of drivers in device functionality

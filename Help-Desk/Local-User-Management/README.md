# Local User Management

## Objective
Pratice creating and managing local Windows users.

## Tools Used
- Windows 11
- VirtualBox
- Command Prompt
- Microsofts GUI

## Commands Used
###cmd
- net user TestUser2 123456 /add
- net user
- net user TestUser2
- net user TestUser2 654321
- net user TestUser2 /active:no 
- net user TestUser2 /active:yes

- How to create local users

# Event Viewer Lab

## Objective
Practice investigating Window logs and authentication events using Event Viewer.

## Tools Used
- Windows 11
- VirtualBox
- Event Viewer

## Task Performed
- Opened Event Viewer
- Explored Window log categories
- Generated failed login attempts
- Investigated authentication events
- Filtered logs using Event IDs
- Reviewed Security logs

## Event IDs Investigated
- 4624 -> Successful login
- 4625 -> Failed login

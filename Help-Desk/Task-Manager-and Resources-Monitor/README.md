# Task Manager and Resource Monitor Lab    

## Objective
Practice investigating and troubleshooting Windows performance issues using Task Manager and Resource Monitor.

## Tools Used
- Windows 11
- VirtualBox
- Task Manager
- Resource Monitor

## Task Performed
- Investigated running processes
- Sorted processes by CPU usage
- Sorted processes by memory usage
- Reviewed startup applications
- Disabled a startup application
- Forced closed application using End Task
- Opened and investigated Resource Monitor
- Reviewed system memory usage

## What I Learned
- How Task Manager is used to troubleshoot slow computers
- How to identify applications consuming high CPU and RAM resources
- How startup applications can impact system performance
- How to force close frozen or unresponsive applications
- How Resource Monitor provides deeper performance annalysis
- Importance of monitoring system resources during troubleshooting
